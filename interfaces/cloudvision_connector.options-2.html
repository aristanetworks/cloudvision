Interface Options

There are a number of options you can supply which limit the scope of the Query. Options are passed as an object, where the key is the name of the option and the value is the value for that option.

There are different combinations of options that are valid, each of them is listed below.

example

// Range query (returns one or more data points)
{
  start: 1505760134000,
  end: 1505760184000,
}

// Point in time query (returns exactly one data point)
{
  end: 1505760184000,
}

// Limit query (returns `versions` + 1 number of data points)
{
  versions: 1,
  end: 1505760184000,
}
