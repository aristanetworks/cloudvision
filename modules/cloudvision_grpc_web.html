fromGrpcInvoke

  • fromGrpcInvoke<Req, Res>(methodDescriptor: MethodDefinition<Req, Res>, options: RpcOptions<Req, Res>, controlFunctions?: Partial<ControlFunctions<Res>>): GrpcSource<Res>
    • Defined in cloudvision-grpc-web/src/grpc/index.ts:34

    Runs grpc.invoke on the GRPC service and converts the response into two observable streams. One is the data stream the other is the message stream. The message stream contains GRPC metadata information, as well as any error messages.

    Type parameters

    • Req: ProtobufMessage

    • Res: ProtobufMessage

    Parameters

    • methodDescriptor: MethodDefinition<Req, Res>

      The GRPC service method definition to be queried.

    • options: RpcOptions<Req, Res>

      Options to pass to eh GRPC call.

    • Optional controlFunctions: Partial<ControlFunctions<Res>>

      (optional) Implementation of the onHeaders, onMessage and onEnd GRPC callbacks. This is where publishing to either the data or message stream is done.

    Returns GrpcSource<Res>

    An object with the properties data and message, which are RXJS Observables that can be subscribed to.
